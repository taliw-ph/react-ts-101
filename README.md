# React and TypeScript 101

## How to run the project

1. Clone the project

```bash
git clone git@github.com:taliw-ph/react-ts-101.git
```

2. Install dependencies

```bash
cd react-ts-101
cd typescript
npm install
```

3. Run the code

```bash
npx ts-node <file-name>.ts

# Example
npx ts-node 00-hello-world.ts
```

## Enjoy the coding!
